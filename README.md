# Obsidian Job Search System
Knowledge Management System to Assist with the Job Search

## Features
This vault provides a robust, easy-to-use KM system to manage your job search. This includes:
* Positions
* Contacts
* Communications
* Companies
* Notes to self

Navigation is provided by a sidebar.

## Requirements
* [Obsidian](https://obsidian.md/) (Free)

## Installation
1. Install Obsidian (if not already installed).
2. Download / Clone the Job_Hunt_Vault folder. Place it wherever you'd like to keep these records.
3. Open the Job_Hunt_Vault folder as a New Vault.
4. The Vault relies on several common Obsidian extensions. 
  These Extensions must be enabled. Either Trust the vault, or enable them manually later.
   * Metadata Menu
   * Dataview
   * Buttons
   * Templater
